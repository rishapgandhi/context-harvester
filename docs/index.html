Capabilities

Collect. Organize. Export.

🖱️ One-Click Capture

Right-click any selected text or press Alt+C. Source URL, page title, and timestamp captured automatically.

📊 Token Awareness

Real-time token counter with model selector — GPT-4 (128k), Claude (200k), Gemini (1M). Never exceed limits.

🏷️ Smart Tagging

Auto-detects content type: code, docs, error, discussion, reference. Organize context intuitively.

↕️ Drag to Reorder

Arrange snippets in the order that makes sense for your prompt. Priority context first.

📤 Structured Export

Export as Markdown with attribution, XML structured prompt, or raw JSON. Copy to clipboard instantly.

🌐 Cross-Browser

Built with WXT framework. Works on Chrome, Firefox, and Edge with a single codebase.
